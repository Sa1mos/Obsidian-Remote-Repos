## Answers

1. Класстерный индекс - это тип индекса, который упорядочивает таблицу на диске.
   Он один, тк данные могут быть отсортированны только одним способом.
2. Не являются.
   В реляционных БД используется система связей с помощью ключей, а в документо-ориентированных - вложенные структуры.
   Хренение данных в реляционных БД реализована в виде блоков фиксированного размера в памяти, когда как документо-ориентированные БД хранят их в виде автономных документов JSON, BSON или XML.
3. `WHERE` используется для фильтрации. Он указывает, что действие должно применяться только к тем записям, которые соответсвуют заданному условию.
```SQL
   SELECT * FROM users WHERE age >= 18;
   
   DELETE FROM users WHERE age > 30
```
